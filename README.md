# Gryph

Основанная на Qt прокси-утилита с ядром, основанным на [Sing-box](https://github.com/SagerNet/sing-box).<br/>
Поддерживает: Windows 11/10/8/7.<br/> 
Скоро будет: Linux, MacOS.<br/>

<img width="1002" height="789" alt="image" src="https://github.com/netward/assets/blob/0ea4b6a34f1dddb86d5f2699ce2da68a61800755/Gryph/mainwindow.png" />

## Поддерживаемые протоколы:

- SOCKS
- HTTP(S)
- Shadowsocks
- Trojan
- VMess
- VLESS
- TUIC
- Hysteria
- Hysteria2
- AnyTLS
- NaïveProxy
- Juicity
- TrustTunnel
- ShadowTLS
- Wireguard
- AmenziaWg
- SSH
- Xray VLESS
- Custom Outbound (Both Sing-box and Xray)
- Custom Config (Both Sing-box and Xray)
- Chaining outbounds
- Extra Core

## Форматы подписки

Поддерживаются различные форматы, в том числе ссылки подписок, различные представления конфигураций Sing-box в формате JSON, формат ссылок v2rayN, а также ограниченная поддержка форматов Shadowsocks и Clash.
